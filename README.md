# Student Performance Analysis Dashboard

## Project Overview

This project analyzes student performance data using Power BI and Excel. The objective is to identify factors that influence academic outcomes and uncover trends across different student groups.

## Objectives

- Analyze student performance in Mathematics, Reading, and Writing
- Examine the impact of gender on academic performance
- Evaluate the effect of test preparation courses
- Explore the relationship between parental education and student outcomes
- Develop an interactive Power BI dashboard for data-driven insights

## Tools Used

- Power BI
- Microsoft Excel
- Microsoft PowerPoint

## Key Insights

- Female students performed better in Reading and Writing.
- Male students showed a slight advantage in Mathematics.
- Students who completed test preparation courses achieved higher scores across all subjects.
- Higher parental education levels were associated with stronger academic performance.
